\' - '
\" - "
\\ - \
\n - перевод на новую строку
\t - символ табуляции (5 пробелов)

Типы данных:
  1. Строковый
    • var user = "Vasya";
    • var user = 'Vasya';
    • var work = 'OOO "Вода"';
    • var work = "OOO 'Вода'";
    • var str = Строка с ' и " в конце;
  2. Числовой
    • var age=35;
    • var dollar=13.5;
  3. Логический
    • var flag=true; (правда, истина)
    • var flag=false; (ложь)

Арифметические операции
+ - * /
** - возведение в степень
% - остаток от деления
